# Security Policy

## Supported Versions

Only the latest major version is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
|   1.0   | :white_check_mark: |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them to the maintainer via email or through GitHub's security advisory feature. We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress.

If you have found a security vulnerability in Cutline OS, please keep it confidential until a fix has been prepared and released. We appreciate your responsible disclosure.
